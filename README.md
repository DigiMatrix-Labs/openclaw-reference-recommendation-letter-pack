# Reference & Recommendation Letter Pack

> Five AI-powered skills for writing professional reference and recommendation letters.

## Skills Included

- **Student Reference Letter Writer** — Writes a formal student reference letter and exports it as a Word or PDF document.
- **Employee Recommendation Letter Writer** — Writes a professional employee recommendation letter and exports it as a Word or PDF document.
- **Tenant Reference Letter Writer** — Writes a landlord reference letter for a rental application and exports it as a Word or PDF document.
- **Character Reference Letter Writer** — Writes a personal character reference letter and exports it as a Word or PDF document.
- **Academic Reference Letter Writer** — Writes a scholarly academic reference letter and exports it as a Word or PDF document.

## Requirements

- [Openclaw](https://openclaw.ai) — AI assistant that runs these skills

## Usage

1. Open Openclaw
2. Load the SKILL.md file for the skill you want to use
3. Follow the steps — Openclaw will ask for your details and produce the file
